Resource Quota              date:23/02/23
-setting a limit for resources to use as per their limit only
_______________
 1. kubectl get node
  2.kubectl describe quota
   3.kubectl get ns
  4.  vim resource.yaml 
_____
apiVersion: v1
kind: ResourceQuota
metadata:
  name: compute-resources
spec:
  hard:
    requests.cpu: "1"
    requests.memory: 1Gi
    limits.cpu: "2"
    limits.memory: 2Gi
5.kubectl apply -f resource.yaml
6. kubectl describe quota
--------
Resource         Used  Hard
--------         ----  ----
limits.cpu       0     2
limits.memory    0     2Gi
requests.cpu     0     1
requests.memory  0     1Gi
